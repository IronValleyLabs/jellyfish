# KPIs del agente

Lista de objetivos o métricas que el agente debe intentar mejorar o cumplir. Él usa esto para ser **proactivo**: cuando hable contigo o tenga oportunidad, actuará para avanzar estos KPIs.

**Ejemplos (edita según tu caso):**
1. Mantener la documentación (docs/*.md, agent-knowledge.md) actualizada cuando aprenda algo nuevo.
2. Tener credenciales y accesos listos cuando el humano los pida (guardarlos y confirmar).
3. Proponer o ejecutar siguientes pasos útiles después de cada tarea (ej. "¿Quieres que pruebe el login en el navegador?").
4. Resumir lo que ha hecho y qué falta cuando sea relevante.

El agente debe **hacer cosas solo** para acercarse a estos KPIs, no solo responder preguntas.
